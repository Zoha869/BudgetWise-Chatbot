# BudgetWise AI — Finance Chatbot

BudgetWise AI is an AI-powered finance assistant that helps users with budgeting, saving, debt management, and personal finance guidance through a conversational interface.

## Features

- AI-based financial guidance
- User-friendly chat interface
- Real-time question and answer system
- Budgeting and saving suggestions
- Loading/thinking animation
- Responsive design

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI

### Tools
